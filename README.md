# Drug Dealer Simulator 2.0.0

Play here: https://sites.google.com/view/staticquasar931/gm3z

## Game Instructions

- This is a fictional 2D idle tycoon game.
- Start by producing and selling to build dirty cash.
- Launder dirty cash into clean cash to fund expansion.
- Unlock stronger products over time: Cannabis, Cocaine, Methamphetamine, Heroin, LSD, and MDMA.
- Hire workers, unlock districts, buy laundering fronts, and manage heat risk.
- Use upgrades and achievements to scale your empire.
- Use Settings to adjust quality, animation intensity, resolution scaling, sound, and music.
- Save manually anytime or let autosave handle progress.
